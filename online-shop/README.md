# BidHub online shop
This is an online shop built with Next.js and TypeScript.  
Users can browse products, search, view details, add items to a cart, and complete a checkout flow.

## Features
- Browse products from API
- View product details
- Search products
- Add/remove items from cart
- Adjust quantity in cart
- Checkout flow with success page
- Contact form with validation

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Noroff API

## API
Data is fetched from the Noroff API:
- GET /online-shop
- GET /online-shop/:id

## Installation
1. Clone the repo:
git clone https://github.com/NoroffFEU/jsfw-2025-v1-idanok.git
2. Go into project folder
cd online-shop
3. Install dependencies:
npm install
4. Run the project:
npm run dev
